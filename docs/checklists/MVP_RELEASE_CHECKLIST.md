# Checklist de release MVP

## Producto

- alcance del release documentado
- copy claro en `es` y `en`
- modulos fuera de alcance no visibles por error

## Seguridad

- permisos revisados
- RLS revisada
- auditoria de acciones relevantes activa

## Calidad

- `typecheck` verde
- `build` verde
- `verify` verde
- smoke e2e verde

## Operacion

- docs actualizados
- Notion actualizado si aplica
- Linear actualizado

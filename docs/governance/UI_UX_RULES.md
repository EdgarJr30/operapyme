# Reglas UI/UX

## Mobile first

- disenar primero en movil
- CTA principal visible
- targets tactiles de al menos 44px

## Formularios

- labels arriba
- feedback de error claro
- estados de guardado visibles

## Listados y paneles

- filtros cortos
- estados vacios utiles
- jerarquia entendible de un vistazo

## Admin

- separar lo comercial de lo operacional
- vistas de auditoria priorizan filtro, trazabilidad y claridad

# PRD de OperaPyme

## Resumen

`OperaPyme` ayuda a pymes a operar su flujo comercial diario desde un backoffice web mobile-first.

## Problema

Muchas pymes operan clientes, cotizaciones, gastos y seguimiento comercial entre WhatsApp, hojas de calculo y documentos dispersos.

## Resultado deseado

Una sola plataforma ligera que permita:

- captar clientes y oportunidades
- construir cotizaciones y proformas rapido
- registrar gastos y deudas
- consultar reportes operativos
- mantener control de permisos y auditoria

## Usuarios iniciales

- dueno o gerente de pyme
- vendedor o asistente comercial
- persona administrativa
- administrador global de plataforma

## Jobs to be done

- "Quiero cotizar rapido desde el telefono y dejar rastro"
- "Quiero ver quien cambio que y cuando"
- "Quiero operar sin volver el sistema un ERP complejo"

## Alcance de fase 1

- fundacion documental
- fundacion segura de backend
- shell del backoffice
- rutas admin reservadas
- baseline de testing

## No objetivos de fase 1

- inventario
- POS
- caja
- facturacion fiscal
- automatizaciones avanzadas
